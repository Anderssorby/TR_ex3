\subsection{The ARMA process}\label{the-arma-process}

A stationary process \(\{X_t\}\) is an \(ARMA(2,1)\) process if the
following relation exists \[
X_t -\phi_1 X_{t-1} - \phi_2 X_{t-2} = Z_t - \theta_1 Z_{t-1}
\] where \$Z\_t - WN(0, \sigma\^{}2) \$
